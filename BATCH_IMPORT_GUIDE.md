# 批量數據導入功能使用指南

## 功能概述

批量數據導入功能允許您快速導入多行數據到數據卡片中，無需逐行手動輸入。您可以從 Excel、Google Sheets 或其他應用程序中複製數據，然後直接粘貼到導入對話框中。

## 使用步驟

### 1. 打開數據管理界面

在儀表板中，找到要更新的數據卡片，點擊卡片上的「管理數據」按鈕。

### 2. 點擊「批量導入」按鈕

在「數據管理」區域中，您會看到一個藍色的「批量導入」按鈕。點擊它打開導入對話框。

### 3. 準備您的數據

數據應以以下格式準備：

```
項目名稱    年度    數值    [備註]
項目名稱    年度    數值    [備註]
...
```

**支持的分隔符**：
- **Tab 鍵**（推薦）：從 Excel 或 Google Sheets 複製時自動使用
- **逗號**：用於 CSV 格式

**數據格式要求**：
- **項目名稱**：必填，任何文本
- **年度**：必填，必須是數字（例如：113、114、115）
- **數值**：必填，必須是數字（支持小數點和負數，例如：100、-50.5）
- **備註**：可選，用於添加額外說明

### 4. 粘貼數據

在導入對話框中，您可以：

- **直接粘貼**：在文本框中粘貼您的數據
- **從剪貼板粘貼**：點擊「從剪貼板粘貼」按鈕自動粘貼

### 5. 預覽數據

點擊「預覽數據」按鈕查看導入結果。系統會：

- 解析您的數據
- 驗證格式
- 顯示將要導入的所有行

如果有錯誤，系統會顯示具體的錯誤信息，您可以返回編輯數據。

### 6. 確認導入

預覽無誤後，點擊「導入數據」按鈕確認導入。新數據會立即顯示在表格中。

### 7. 保存更改

導入完成後，點擊「保存數據」按鈕將更改保存到數據庫。

## 使用示例

### Excel 複製粘貼

1. 在 Excel 中選擇數據（包括項目名稱、年度、數值列）
2. 按 Ctrl+C（或 Cmd+C）複製
3. 在導入對話框中粘貼

**Excel 示例**：
```
總登記數貓    115    25000
新登記犬      115    2500
總登記數犬    115    30000
新登記貓      115    5500
```

### CSV 格式

```
項目A,113,100
項目B,113,200
項目C,114,150
```

### 帶備註的數據

```
總登記數貓    115    25000    2024年新增
新登記犬      115    2500     預估值
總登記數犬    115    30000    已驗證
```

## 數據驗證規則

系統會自動驗證您的數據。以下情況會導致導入失敗：

| 驗證項目 | 要求 | 示例 |
|---------|------|------|
| 項目名稱 | 不能為空 | ✅ 總登記數貓 |
| 年度 | 必須是數字 | ✅ 115 ❌ 115年 |
| 數值 | 必須是數字（支持小數和負數） | ✅ 100 ✅ -50.5 ❌ abc |
| 行格式 | 至少包含 3 個欄位 | ✅ 項目 115 100 ❌ 項目 115 |

## 常見問題

### Q: 導入會覆蓋現有數據嗎？

**A**: 不會。導入只會添加新行到表格中。如果您想替換現有數據，需要先手動刪除舊行。

### Q: 可以導入多少行數據？

**A**: 理論上沒有限制，但建議一次導入不超過 1000 行以保持良好的性能。

### Q: 導入失敗了怎麼辦？

**A**: 檢查錯誤信息，通常是因為：
- 年度不是數字
- 數值包含非數字字符
- 缺少必要欄位

修正後重新嘗試。

### Q: 可以導入帶有特殊字符的項目名稱嗎？

**A**: 可以。項目名稱支持任何文本，包括中文、特殊字符等。

### Q: 導入後可以撤銷嗎？

**A**: 導入後，您可以手動刪除導入的行。如果需要完全撤銷，請刷新頁面（未保存的情況下）或聯繫管理員。

## 最佳實踐

1. **驗證數據**：導入前在源應用中驗證數據的準確性
2. **備份**：在進行大量導入前，建議先備份現有數據
3. **分批導入**：對於大量數據，考慮分批導入以便於檢查
4. **使用備註**：為重要數據添加備註，便於後續追蹤
5. **定期保存**：導入完成後立即保存，避免數據丟失

## 技術細節

- **前端驗證**：所有數據驗證在前端進行，確保快速反饋
- **支持格式**：Tab 分隔符（推薦）和逗號分隔符
- **數據類型**：年度和數值自動轉換為正確的數據類型
- **錯誤處理**：詳細的錯誤信息幫助用戶快速定位問題
